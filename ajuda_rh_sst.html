Cadastro de Funcionários
Como cadastrar um novo funcionário? ▼
1. Acesse Gestão RH & SST → aba Funcionários
2. Clique em + Novo Funcionário
3. Preencha os campos obrigatórios: Nome, CPF, Cargo, Setor, Data de Admissão
4. Campos opcionais: telefone, e-mail, salário, empresa, filial, matrícula, PIS
5. Clique em Salvar

O funcionário será criado com status ativo automaticamente.
Como editar os dados de um funcionário? ▼
Na tabela de funcionários, clique no ícone ✏️ na coluna Ações.
O formulário abrirá preenchido com os dados atuais. Altere o que precisar e clique em Salvar.

Importante: Toda alteração é registrada no log de auditoria (quem alterou, quando, dados antes/depois).
O que é o Motor Lógico? ▼
O Motor Lógico responde automaticamente: "Este funcionário pode trabalhar hoje?"

Ele verifica:
• ASO válido — se o exame está dentro da validade e o resultado é "Apto"
• Treinamento de Integração — se foi realizado
• Treinamentos NR — se todos estão em dia
• Restrições — se há restrições ativas (peso, altura, etc.)
• Status — se o funcionário não está afastado, em férias ou inativo

Se tudo estiver OK: APTO
Se qualquer item falhar: BLOQUEADO (mostra os motivos ao passar o mouse)
ASO — Exames Ocupacionais
Como registrar um ASO? ▼
1. Aba ASO → + Novo ASO
2. Selecione o funcionário
3. Preencha: Tipo (Admissional, Periódico, Retorno, Mudança de Função, Demissional), Data do Exame, Data de Validade, Resultado
4. Opcionalmente: médico responsável e CRM

O sistema calcula automaticamente o status: Válido, Vence em 30d ou Vencido.
O que acontece quando um ASO vence? ▼
• O funcionário aparece como BLOQUEADO no Motor Lógico
• O Dashboard SST mostra alerta vermelho
• O ASO aparece na lista de Pendências Críticas
• O KPI "% Em dia" é atualizado automaticamente

Prazo legal: ASOs periódicos devem ser renovados conforme NR-07 (geralmente a cada 12 meses, ou 6 meses para menores e expostos).
Treinamentos NR
Como registrar um treinamento? ▼
1. Aba Treinamentos → + Novo Treinamento
2. Selecione o funcionário e o tipo de NR (NR-06, NR-10, NR-12, NR-35, NR-36, CIPA, Integração, etc.)
3. Preencha data de realização, validade e carga horária
4. Opcionalmente: nome do instrutor

NR-36: Obrigatório para todos que trabalham em abate e processamento de carnes.
EPIs — Equipamentos de Proteção
Como registrar entrega de EPI? ▼
1. Aba EPIs → + Registrar Entrega
2. Selecione o funcionário
3. Informe: Nome do EPI, CA (Certificado de Aprovação), Data de Entrega, Validade, Quantidade

Dica: O número do CA é obrigatório pela NR-06 e deve constar na ficha de entrega.
Programa Saúde e Equilíbrio
Como funciona o Programa de Saúde Mental? ▼
O programa atende à NR-1 atualizada (riscos psicossociais). Funciona em 3 vias:

• Demanda espontânea — o colaborador solicita voluntariamente
• Indicação da liderança — baseada em critérios objetivos
• Gatilho automático — o sistema identifica sinais (faltas > 3/mês, retorno de afastamento)

O que é registrado: Data, comparecimento, classificação de risco (Baixo/Moderado/Alto) e recomendação geral.
O que NÃO é registrado: Conteúdo clínico das sessões (sigilo absoluto).
O que significam os níveis de risco? ▼
Nível 1 — Baixo: Acompanhamento opcional. Colaborador estável.
Nível 2 — Moderado: Monitoramento contínuo. Agendar acompanhamento.
Nível 3 — Alto: Intervenção necessária. Alerta automático para RH. Possível encaminhamento médico.

Classificação realizada pela psicóloga responsável após cada atendimento.
Desligamentos e Turnover
Como registrar um desligamento? ▼
1. Aba Desligamentos → + Registrar Desligamento
2. Selecione o funcionário, data, tipo (Pedido, Demissão, Término) e categoria da causa
3. Preencha a Entrevista de Desligamento (5 perguntas padrão)
4. Ao salvar, o funcionário é automaticamente marcado como inativo

Categorias de causa:
• Admissão — não adaptou, expectativa errada, falta de habilidade
• Operacional — liderança, ambiente, carga, conflito
• Financeiro — salário, benefícios
• Pessoal — família, mudança, saúde
Como o sistema ajuda a reduzir o turnover? ▼
O Dashboard SST cruza dados automaticamente:

• Turnover de curto prazo: Funcionários que saíram com menos de 90 dias → indica erro na admissão/integração
• Turnover por setor: Identifica setores com gestão problemática
• Distribuição por causa: Mostra se o problema é admissão, operacional, financeiro ou pessoal
• Tempo médio até desligamento: Indica em que fase o funcionário desiste

Use a entrevista de desligamento sempre — é a fonte mais rica de dados para prevenção.
Pagamentos e Exportação
Como exportar a lista para o financeiro? ▼
1. Acesse Gestão de Pagamentos → aba Exportar Listas
2. Escolha o tipo: Itaú, Sicredi, PIX, Cheque, Consolidada ou Pendências
3. Clique em Excel ou CSV

O arquivo gerado contém: nome, matrícula, CPF, empresa, filial, setor, cargo, forma de pagamento, banco, agência, conta, tipo conta, chave PIX, titular, CPF titular, status de validação.

Toda exportação é registrada no log (quem exportou, quando, tipo, filtros).
Auditoria e Segurança
O que é registrado no log de auditoria? ▼
Toda ação no sistema gera um registro com:
• Quem fez (nome e ID do usuário)
• Quando (data e hora exatas)
• O que fez (criar, editar, excluir)
• Dados antes e depois da alteração

Exemplos: alterou cargo → log · entregou EPI → log · mudou risco → log · exportou lista → log
O sistema está adequado à LGPD? ▼
O sistema foi construído com as seguintes proteções:

• Base legal: Relação de emprego (Art. 7, V) e Proteção à saúde (Art. 7, VIII)
• Minimização: Saúde mental NÃO armazena conteúdo clínico
• Acesso restrito: RLS por perfil (colaborador, líder, RH, SST, psicóloga)
• Auditoria: Log completo de quem acessou/alterou dados
• Retenção: Dados trabalhistas = 5 anos CLT / ASO = 20 anos NR-07
